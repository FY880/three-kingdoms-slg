# 三国 SLG · 项目交接说明（手机端 → 电脑端）

> 本说明由手机端云端会话生成，供电脑客户端助理接手继续开发。
> 生成时间：2026-07-09

## 1. 项目概况

单玩家三国 SLG（策略战棋）。架构上**纯逻辑层 `src/` 与表现层 `index.html` 分离**：

- 逻辑层零 DOM、UMD 双导出，浏览器 / Node 都能复用。
- 配置数据走 CSV 热加载（`data/*.csv`），可在不碰代码的情况下调数值。

## 2. 目录结构与关键文件

| 路径 | 作用 |
|------|------|
| `src/{util,data,combat,diplomacy,scenario,rules}.js` | 纯逻辑引擎（6 个模块，UMD 模式） |
| `data/{terrain,weather,season,generals,formations}.csv` | 配表：地形 / 天气 / 季节 / 武将 / 阵型 |
| `index.html` | Canvas 移动端 Demo（`fetch` 加载 CSV，需 http 服务器） |
| `dist/index.standalone.html` | **单文件离线版**：内联全部 JS + CSV，零服务器，手机/电脑双击即玩 |
| `tools/build-standalone.js` | 生成单文件离线包 |
| `tools/build-bundle.js` / `engine-adapter.js` | 引擎 bundle / 适配层 |
| `tools/deploy-gh-pages.sh` | GitHub Pages 一键部署（需 `gh` 登录） |
| `tests/{smoke,balance,unit}.js` + `balance_report.md` | 冒烟 / 平衡 / 单测 |
| `三国SLG_游戏设计文档_GDD.md` | 设计文档 |

## 5. 如何运行 / 出包 / 部署

- **本地试玩（最简单）**：双击 `dist/index.standalone.html`（零服务器）。
- **改完源码出包**：`node tools/build-standalone.js` → 重新生成 `dist/index.standalone.html`。
- **起本地服务器看 `index.html` 原版**：`python3 -m http.server` 后浏览器开 `index.html`（`fetch` 需 http 环境，不能直接 `file://` 打开）。
- **跑测试**：`node tests/smoke.js`、`node tests/balance.js`。
- **发布网页（需先 `gh auth login`）**：`bash tools/deploy-gh-pages.sh`。
