## 一、美术与视觉（权重 25%）

### 当前状态
| 项目 | 我们的 | 率土 | 谋定天下 | 差距 |
|------|--------|------|----------|------|
| 地图渲染 | Canvas 纯色填充 + 文字字形 | 高清地形贴图 + 光照 | 实时 3D 渲染 | 🔴🔴🔴 |
| 武将形象 | 无立绘/头像，纯文字名 | 卡牌式立绘（每将一图） | 动态卡牌+Live2D | 🔴🔴🔴 |
| 城市表现 | 「城」字文本 + 蓝底 | 3D 城池模型 + 动态旗帜 | 3D 城市 + 光影 | 🔴🔴🔴 |
| 战斗表现 | 闪光边框+浮动文字「占领」 | 实时战斗动画(30秒) | 电影级战斗演出 | 🔴🔴🔴 |

| UI 设计 | 深灰面板+基础圆角 | 古风毛笔质感+金线装饰 | 新中式极简高级 | 🔴🔴 |
| 特效反馈 | 攻击闪光(白色/红色边框) | 技能特效动画链 | 粒子+屏幕震动+慢镜 | 🔴🔴🔴 |
| 过渡动画 | 无（面板直接出现） | 滑入/淡入/弹性动画 | 流畅转场系统 | 🔴🔴 |
| 小地图 | **完全缺失** | 右下角缩略图+视野框 | 同左+实时标记 | 🔴🔴🔴 |
| 加载画面 | 无（瞬间显示） | 剧情插画加载屏 | 品牌CG | 🟡 |

### 美术差距总结
**整体评分：15/100 → 目标 90**
- 最致命缺失：**无任何图像资源**（武将/城市/技能/宝物全是文字）
- 单文件架构限制：无法引入大量图片 → 但可用 CSS 渐变/Canvas 绘制提升观感

### 可执行改进（无需外部素材）
1. **小地图** — Canvas 缩略图 + 玩家位置标记 + 可点击跳转
2. **武将色卡** — 用 CSS 渐变生成"卡牌感"背景（按势力色/品质色）
3. **攻击特效升级** — 多层闪光 + 屏幕抖动增强 + 击中粒子
4. **天气视觉强化** — 更多粒子 + 全屏色调叠加加强
5. **UI 微动效** — 面板 slideUp 弹入 + 按钮 press 反馈 + 数字滚动
6. **地形纹理化** — Canvas pattern fill（草地噪点/水面波纹）代替纯色
7. **资源飘字** — 产出时数字从地块飘起消失（+12粮 / +3玉）

| 战斗回放 | 无 | 可回看战斗过程 | 同左 | 🟡 |

### 操作差距总结
**评分：20/100 → 目标 90**

### 可执行改进
1. **交互式新手引导** — 5步引导（关Intro→认地图→点领地→推进时辰→出征），每步遮罩+高亮目标+下一步按钮
2. **小地图** — 右下角 120×90 Canvas，绘制全地图缩略，蓝框=当前视口，绿点=玩家主城，可点击跳转
3. **「回主城」按钮** — 点击后相机平滑飞回玩家起始格并设合适 zoom
4. **信息卡片** — 长按/点击地块弹出详情浮层（地形/归属/守军/产量/可行动作列表）
5. **设置面板** — 音效开关（占位）、重开游戏、关于/版本号
6. **Toast 升级** — 从底部滑入 + 自动消失 + 可堆叠
7. **操作确认** — 出征前二次确认（避免误触）、放弃领土确认

| 系统 | 我们的 | 率土 | 谋定天下 | 差距 |
|------|--------|------|----------|------|

| 武将进阶 | **无**（只有星级标签） | 3星→4星→5星 进阶（消耗同名卡+材料） | 同左 | 🔴🔴🔴 |

| 宝物装备 | 有（但仅名称+数值加成，无可视化） | 武器/防具/马匹/宝物四槽+套装效果 | 同左+外观 | 🔴🔴 |
| 科技树 | **无** | 同盟科技（军事/内政/补给三条线） | 个人科技树 | 🔴🔴🔴 |
| 建筑/内政 | **无** | 主城建筑队列（政厅/军营/工坊等） | 同左+委任系统 | 🔴🔴🔵 |
| 人口系统 | **无** | 人口→征兵上限→税收 | 同左+流民 | 🔴🔴 |
| 兵种质量 | **无**（兵力只是数字） | 兵种分级（预备兵/民兵/正规军/精锐） | 同左+兵装 | 🔴🔴 |
| 经济深度 | 5种资源线性产出 | 4资源+铜钱+玉，多层级消耗循环 | 同左+辎重 | 🔴🔴 |
| 资源交易 | **无** | 盟内/跨盟交易 | 同左+黑市 | 🟡 |
| 难度曲线 | AI 固定强度 | 随时间/AI扩张变强 | 动态难度 | 🟡 |

### 数值差距总结
**评分：18/100 → 目标 90**

### 可执行改进
1. **武将经验/等级系统** — 出征获得经验 → 升级 → 统/武/智/政 各 +N%
2. **武将进阶** — 消耗同名武将卡或材料提升星级上限
3. **科技树** — 用玉解锁科技（军事线：攻击/防御；经济线：产量/效率）
4. **建筑系统** — 简化版：政厅(人口上限)、军营(兵力上限)、工坊(建造速度)
5. **兵种质量** — 预备兵(×0.7)→ 正规军(×1.0)→ 精锐(×1.3)，消耗资源升级
6. **经济循环** — 给玉一个持续消耗出口（科技/进阶/招募），形成闭环
7. **AI 动态难度** — 随回合增加 AI 增援频率和守军质量

| 奖励领取 | **无** | 任务完成后手动领奖 | 同左+一键领取 | 🔴🔴 |
| 目标指引 | 顶栏 objBar 一行文字 | 侧边栏详细进度+剩余条件 | 同左 | 🔴🔴 |
| 首充/七日 | **无** | 七日目标+累充奖励 | 同左 | 🔴🔴 |
| 里程碑 | **无** | 首次占领X地/首次战胜/首次同盟 | 同左 | 🔴🔴 |

### 任务差距总结
**评分：5/100 → 目标 90**

### 可执行改进
1. **日常任务系统** — 每天生成 5 个任务：
   - 【出征】进攻 X 次（奖励：币+粮）
   - 【扩张】占领新领土 X 格（奖励：木+铁）
   - 【练武】推进时辰 X 次（奖励：经验书）
   - 【富国】资源总产出达 X（奖励：玉）
   - 【称霸】占领州府 X 座（奖励：宝物碎片）
2. **成就系统** — 分类记录：
   - 战斗类：首杀/百胜/零损胜利/以少胜多
   - 领土类：占领10/50/100格、控制整州
   - 收集类：拥有X武将、集齐某势力全员
   - 特殊类：单回合攻占5格、不损失一兵统一
3. **七日目标** — 登录第1~7天的递进奖励（模拟新手期激励）
4. **任务面板** — 底部新增「任务」tab 或在养成页签内
